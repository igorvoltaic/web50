vault secrets enable ssh
vault write ssh/roles/root_otp \
    key_type=otp \
    default_user=root \
    cidr_list=0.0.0.0/0

vault can be opened using token in user's home dir:

vault ssh root@localhost (you could run it just out of user's shell, no 'vault login' step needed)
